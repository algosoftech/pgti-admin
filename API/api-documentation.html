🔐 Authentication Requirements

  • Auth Token: Required for protected endpoints. Send in header as authToken or query parameter authToken
  • API Key: Required for front-end endpoints. Send in header as api-key or query parameter api-key
  • Optional Auth: Some endpoints support both authenticated and guest users
