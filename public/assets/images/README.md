# Premium Golf Login Image Assets

Place the production golf course image here:

- `golf-login-course.webp`

Recommended image requirements:

- Format: WebP or AVIF
- Size: around 1920x1080 or 2400x1350
- Style: realistic premium golf course, morning or golden-hour lighting
- Composition: playable fairway/green visible across lower half of the frame
- Licensing: client-owned or properly licensed, no watermarks
- Optimization: compressed for web delivery

If this image is missing, the login screen uses the layered CSS course fallback.
